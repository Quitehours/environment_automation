# VIM Guide Script

A colorful and comprehensive Vim cheat sheet as a shell script.

## Quick Start

1. **Make the script executable:**
   ```bash
   chmod +x vim_guide.sh
   ```

2. **Run the script:**
   ```bash
   ./vim_guide.sh
   ```
